# Inkspark Digital Blog

Inkspark Digital Blog is a simple static blog built with HTML and CSS. The site contains different categories of content, from travel to tech, and is intended to be lightweight and easy to host.

## Features
- Static HTML/CSS site — no build step required
- Simple, easy-to-edit files
- Suitable for local development and easy hosting on static hosts (GitHub Pages, Netlify, Vercel, etc.)

## Prerequisites
- A web browser (Chrome, Firefox, Edge, Safari, etc.)
- Git (optional, for cloning the repo)
- Optional: Python 3 or Node.js if you want to run a local static server
## Quick setup (open locally)
Option A — Open directly in a browser
1. Download or clone the repository:
   - Clone: `git clone https://github.com/Kyalo-ai/Inkspark-blog.git`
2. Open `index.html` in your browser (double-click the file or use “Open file” in your browser).
